# README


* System dependencies:
  - ruby 2.7.1
  - bundler
  - yarn
  - sqlite3
  - cron
  - (optional)rbenv

* Deployment instructions:
  * bundler install
  * yarn install
  * rails db:migrate 
  * rails server
  * (optional cron tasks) whenever --update-crontab 

First created user will be an admin.
